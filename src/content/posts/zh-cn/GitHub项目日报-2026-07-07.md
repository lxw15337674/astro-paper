## 1. [MadsLorentzen/ai-job-search](https://github.com/MadsLorentzen/ai-job-search)

- 项目总结：这是一个基于 Claude Code 的求职自动化框架，用结构化流程帮助用户筛选岗位、定制简历与求职信并准备面试。
- 技术栈：TypeScript / Claude Code CLI / Python 3.10+ / Bun / LaTeX（lualatex、xelatex）/ pdftotext（可选）
- 使用场景：适合希望把岗位搜索、匹配评估和材料生成流程标准化的个人求职者使用。
- Stars：10.8k
- Forks：3.6k

## 4. [ruvnet/RuView](https://github.com/ruvnet/RuView)

- 使用场景：适合智能家居、房间占用检测、非接触式生命体征监测和实验性边缘感知场景。
