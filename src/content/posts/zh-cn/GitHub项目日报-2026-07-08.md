## 2. [ruvnet/RuView](https://github.com/ruvnet/RuView)

- 项目总结：这是一个基于 WiFi CSI 的边缘感知平台，项目用于在无摄像头和无穿戴设备条件下做存在检测、生命体征监测、活动识别与空间感知。
- 技术栈：Rust / ESP32-S3、ESP32-C6 / Raspberry Pi / MQTT、Matter、Home Assistant / Hugging Face 模型 / Candle / three.js
- 使用场景：适合需要隐私优先的室内感知场景，如智能家居、房间占用监测、睡眠与跌倒告警，以及基于边缘设备的实验性 WiFi 感知开发。
- Stars：79.1k
- Forks：10.6k
- 今日新增 Stars：799

## 3. [TencentCloud/TencentDB-Agent-Memory](https://github.com/TencentCloud/TencentDB-Agent-Memory)

- 项目总结：这是一个面向 AI Agent 的本地记忆系统，用分层长期记忆与 Mermaid 符号化短期记忆来降低上下文开销并保留可追溯性。
- 技术栈：TypeScript / SQLite、sqlite-vec / OpenClaw 插件 / Hermes Gateway / Markdown、JSONL、Mermaid
- 使用场景：适合需要让 Agent 在长会话、跨任务或跨轮次中记住用户偏好、任务状态与历史证据的本地化部署场景。
