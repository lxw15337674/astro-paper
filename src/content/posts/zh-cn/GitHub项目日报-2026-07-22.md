## 1. [koala73/worldmonitor](https://github.com/koala73/worldmonitor)

- 项目总结：World Monitor 是一个将新闻聚合、地缘政治监测、基础设施追踪与金融信息整合到地图界面的实时全球态势感知仪表盘。
- 技术栈：TypeScript、Vite、globe.gl、Three.js、deck.gl、MapLibre GL、Ollama、Transformers.js、Tauri 2、Rust、Redis、Vercel Edge Functions
- 使用场景：适合研究、监测和开发团队搭建可自托管的多源信息监控界面，也可通过 REST API、MCP、CLI 和 SDK 接入脚本或智能体。

- Forks：10.5k
- 今日新增 Stars：4.1k

## 2. [ruvnet/RuView](https://github.com/ruvnet/RuView)

- 项目总结：RuView 使用带有信道状态信息（CSI）的 WiFi 信号进行无摄像头空间感知，可识别人 presence、活动、运动及部分生命体征。
- 技术栈：Rust、ESP32-S3/ESP32-C6、CSI、Candle、时序信号处理、脉冲神经网络、Cognitum Seed、Home Assistant MQTT、Matter、Tauri 2
- 使用场景：适合边缘计算、智能家居、空间占用和研究场景，但高级感知能力依赖 CSI 硬件，项目 README 将其标为测试中的软件且 API 与固件可能变化。
- Stars：83.8k
- Forks：11.2k
- 今日新增 Stars：741

## 4. [schollz/croc](https://github.com/schollz/croc)

- 项目总结：croc 是一个通过中继在任意两台计算机之间传输文件、目录和文本的命令行工具，并提供端到端加密、断点续传和跨平台支持。
- 技术栈：Go、CLI、中继服务、PAKE、IPv6/IPv4、Tor SOCKS5 代理、Docker、Homebrew、Scoop、Termux
