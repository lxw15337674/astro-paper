## 1. [block/buzz](https://github.com/block/buzz)

- 项目总结：Buzz 是一个可自托管的协作工作空间，将人类、AI 代理、工作流和 Git 事件放入同一套可审计事件日志中。
- 技术栈：Rust / Nostr relay / Axum / WebSocket / REST / PostgreSQL / Redis / Tauri / React / Docker
- 使用场景：适合需要让开发者与 AI 代理协同处理代码、评审、CI 工作流和项目记忆的团队。
