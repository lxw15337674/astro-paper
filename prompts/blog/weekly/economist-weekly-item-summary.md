# 任务

根据一篇《经济学人》文章的 EPUB 证据生成中文文章级摘要。你是整期导读的上游步骤，输出会被后续编辑模型使用。

- 刊期归档日期：{date}
- 文章编号：{rank}

# 输入证据

```md
{article_text}
```

# 输出格式

只输出一个合法 JSON 对象，不要输出 Markdown、代码围栏或任何解释。必须严格使用输入文章编号：

```json
{
  "rank": {rank},
  "one_sentence_summary": "40—80 字，一句话说明文章讨论什么和结论是什么。",
  "core_point": "100—180 字，明确核心论点、关键因果或分歧。",
  "content_summary": "220—400 字，完整交代背景、主要事实、论证过程、结论和必要边界，不复刻正文。"
}
```

三个字段不得换一种说法重复同一段内容。不得编造作者、引语、数字、立场、链接或输入中不存在的事实；不得提及 EPUB、模型或提示词。
