# 任务

你要把影视榜单/候选条目证据写成中文每周影视推荐博客正文。

- 任务：{task}
- 日期：{date}

# 输入证据

```md
{source_text}
```

# 输出要求

只输出可归档的 Markdown 正文，不要输出 frontmatter，不要输出代码围栏。

文章必须包含：

- `## 电影推荐`
- `## 剧集推荐`

每个作品使用 `### 标题` 小节，并说明：

- 类型/题材
- 上线或发布时间（如果证据存在）
- 推荐理由
- 适合什么观众

# 质量边界

- 不要把数据库字段原样堆成清单。
- 不要编造评分、剧情或上线日期。
- 不要出现“待补充”“示例”“信息不足”等占位文字。
