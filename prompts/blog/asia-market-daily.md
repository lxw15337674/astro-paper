# 任务

你要为 Astro Paper 博客生成一篇中文亚洲市场日报正文。

- 任务：{task}
- 日期：{date}

# 输入证据

下面是脚本抓取的 A股、港股指数、成交与板块结构。只能依据这些证据写作；不要补不存在的行情、个股、政策或宏观数据。

```md
{source_text}
```

# 输出要求

只输出可归档的 Markdown 正文，不要输出 frontmatter，不要输出代码围栏。

文章必须包含并优先保持以下结构：

- `## 总结`
- `## A股`
- `## A股行业板块`
- `## 港股`

# 总结规则

- `## 总结` 必须放在正文最上面。
- 总结只概括可复核的指数、成交、板块结构、休市状态和数据边界。
- 如果某个市场当日未产生或未获取到完整常规交易数据，必须说明，不要用最近一个交易日假装当天开盘。

# 固定文本保留

如果输入证据中出现以下固定文本，输出时必须原样保留，不能改写成行情判断：

- `A股当日未产生完整常规交易数据，本节不做指数表现与市场结构判断。`
- `港股当日未产生完整常规交易数据，本节不做指数表现与市场结构判断。`

如果输入证据写的是“未获取到完整常规交易数据”，也必须保留该数据边界，不要补行情。

# 写作风格

- 中文、清晰、克制，像给投资者和工程读者看的自动化市场简报。
- 优先解释“数据说明什么”和“哪些不能从当前数据推出”。
- 不要写鸡汤、标题党或无法由证据支持的交易建议。
- 禁止出现“建议关注”“值得关注”“继续关注”“后续关注”“最看好”“赚钱点子”“操作”“布局”“机会”“交易建议”“投资建议”等动作导向语言。
- 不要出现“待补充”“示例”“稍后更新”等占位文字。
